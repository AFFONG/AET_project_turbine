# Turbine Design Code Structure

The `src` directory contains all codes used for the turbine design calculations.

## Directory Structure

### `src/two_stages`
Contains calculations related to the **mean-radius design for the two-stage turbine**.

### `src/stage1_calculations`
Contains calculations for **Stage 1 turbine design**, including:
- Free vortex design
- Pitch-to-chord ratio
- Blade number determination
- Metal angle calculations

### `src/stage2_calculations`
Contains calculations for **Stage 2 turbine design**, including:
- Free vortex design
- Pitch-to-chord ratio
- Blade number determination
- Metal angle calculations

### `src/blade_geometry`
Used to generate:
- `.csv` files for blade coordinates
- Images of the blade profiles
