src contains all the codes for turbine design 
src/two_stages: contains calculations related to mean-radius design
src/stage1_calculations: contains calculations related to the first stage (free vortex design, pitch-chord ratio, blade number, metal angles)
src/stage2_calculations: contains calculations related to the second stage. (free vortex design, pitch-chord ratio, blade number, metal angles)
src/blade_geometry: used to generate .csv and images for blade profiles
